MinibloxV4

by ee6-lang

About

MinibloxV4 is a lightweight reverse-engineering and scripting client for the game Miniblox. Built for performance and modifiability, it's ideal for developers and hackers exploring client internals.

Features

  • Injectable client with bypasses
  • Modular scripting support
  • UI overlay for enabling modules
  • Example scripts and utilities

Installation

Clone the repo and follow the instructions:

git clone https://codeberg.org/ee6-lang/MinibloxV4.git
cd MinibloxV4
# Follow included README for injection/setup
